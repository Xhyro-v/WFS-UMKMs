## What is this?
**WFS SMEs/UMKM** It is a full-stack website for demonstrating a website concept targeting SMEs/UMKM, and also serves as a learning project.

---

## ✨ Features

- 🔐 Admin authentication system
- 📰 Content management system (news, events, promotions)
- 🖼️ Gallery management
- 🍽️ Menu management
- 📊 Centralized website dashboard for full content control

---

## 🛠️ Tech Stack

###  Programming Language
- Python 3.10+
- HTML5
- CSS
- JavaScript

### Backend Framework
- FastAPI
- Starlette (ASGI foundation)
- Uvicorn (ASGI server)

### Database
- PostgreSQL
- SQLAlchemy (ORM)
- Alembic (database migration)

### Validation & Serialization
- Pydantic

### Template Engine (Frontend SSR)
- Jinja2

### Utilities & Tools
- python-dotenv (environment variables)
- python-multipart (form & file upload handling)
- Click (CLI utilities)

### Database Driver
- psycopg (PostgreSQL driver)
